## Layer 1 — Summary

A first-class, model-agnostic, local-first knowledge system for Claude. Content-agnostic and consumer-agnostic. Exposed via MCP as native tools. Retrieval by meaning, not keyword. Not a documentation tool — a persistent, queryable memory layer.

## Layer 2 — Full Detail

Build a first-class, model-agnostic knowledge system for Claude to interact with natively.

Core properties:
- Content-agnostic: any kind of knowledge can be stored
- Consumer-agnostic: not designed for a specific user or team
- Claude-native: exposed via MCP so Claude can use it as tools in any session
- Local-first: persists across restarts, no cloud dependency required
- Semantic: retrieval by meaning, not keyword

The system is not a documentation tool. It is a persistent, queryable memory layer.
